# Visão Geral do Projeto

## Objetivo
Documentar um projeto de BI em Power BI com dados extraídos do **SQL Server** e KPIs calculados em **DAX** no modelo. O foco é estruturar governança, catálogo de KPIs, páginas e insights com base nos prints do dashboard.

## Escopo
- Documentação do modelo dimensional
- Catálogo de KPIs (DAX)
- Descrição de páginas e visuais
- Insights acionáveis por página
- Apresentação executiva (PPTX)

## Premissas
- Os dados são obtidos do **SQL Server (extração/refresh via Power BI)**.
- Os KPIs são **calculados em DAX** (não em SQL).
- Descrições e insights se baseiam nos prints disponíveis.
